# Image Design Language Registry

Read this index first, then open only the selected design-language spec files.

| ID | Best For | Visual Traits | Best Layouts | Avoid Layouts | Information/Text Strategy |
|---|---|---|---|---|---|
| `editorial-artifact` | technical explainer, product note, model update | tactile artifacts, crop, evidence table | artifact-board, product-specimen, receipt-comparison | cinematic-still | headline + evidence labels |
| `data-poster` | metrics, benchmarks, comparisons | large numbers, structured fields, proof marks | big-number, split-field, flow-diagram | full-bleed-poster | exact metric hierarchy |
| `field-notes` | research notes, observations, checklist-like ideas | paper, note marks, specimen notes | artifact-board, receipt-comparison, flow-diagram | full-bleed-poster | checklist-style facts |
| `image-led-magazine` | premium carousel, editorial visual story | strong crops, image rhythm, magazine pacing | contact-sheet, product-specimen, full-bleed-poster | receipt-comparison | headline + captions/labels |
| `newsroom-poster` | news, launch, incident, public update | deadline energy, stamp marks, high contrast | full-bleed-poster, big-number, contact-sheet | product-specimen | bold headline + date/fact |
| `product-catalog` | product/tool/model specimen, feature sheet | controlled studio object, catalog crop | product-specimen, split-field, contact-sheet | receipt-comparison | title + feature tags |
| `cinematic-still` | mood cover, story opener, dramatic implication | scene, light, atmosphere, subject dominance | full-bleed-poster, contact-sheet | receipt-comparison, flow-diagram | title + one claim |
| `swiss-poster` | bold thesis, simple comparison, graphic hook | geometric fields, scale contrast, restrained palette | full-bleed-poster, split-field, big-number | artifact-board | very short, very large text |

Use `schema.md` for the required fields and extension rules.
