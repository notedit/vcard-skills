# Social Feed Patterns

Use this when the card set targets mobile carousel feeds, Chinese social image posts, or image-first sharing contexts.

## Platform Defaults

- Ratio: `4:5`.
- Card count: 4-7 for explainers, 2-4 for visual-led but still information-bearing posts, 1-3 for cover variants.
- Cover decides the swipe. It uses only the user title plus one generated guiding sentence. The title must be the largest focus; the guiding sentence should be short, insightful, restrained, and make the reader want the remaining cards.
- Middle cards should be save-worthy: checklist, contrast, steps, proof, object notes, or concrete examples.
- Ending can be a summary, decision rule, or interaction prompt.

## Content Types

| Type | Best Arc | Visual Direction |
|------|----------|------------------|
| 干货分享 | hook -> context -> method -> checklist -> takeaway | notes, artifacts, annotated object |
| 避坑指南 | warning -> mistake -> consequence -> fix -> checklist | warning label, receipt, stamp, torn note |
| 测评对比 | verdict -> A/B proof -> use case -> recommendation | split scene, product specimen, side-by-side object |
| 教程步骤 | result -> materials -> steps -> checks -> save | process stills, numbered objects, desk setup |
| 观点评论 | thesis -> contrast -> evidence -> implication -> question | editorial poster, symbolic still, cropped object |
| 生活/种草 | scene -> problem -> discovery -> detail -> feeling | lifestyle still, product crop, warm object photography |

## Hook Types

Use only truthful hooks:

- pain: "别再...", "踩坑后才知道..."
- identity: "给独立开发者", "打工人..."
- curiosity: "真正拉开差距的是..."
- benefit: "少走 3 个弯路", "交付前多看这一眼"
- contrast: "不是 A，而是 B"

Avoid fake-perfect claims like "99% 的人不知道" unless sourced.

## Swipe Arc

Common 5-card arc:

1. Cover: user title + one guiding sentence, with title as the largest visual focus.
2. Context: why this matters now.
3. Mechanism: what actually happens.
4. Checklist/proof: 3-5 useful observations.
5. Takeaway: action rule plus lightweight interaction prompt.

Each card should give a reason to swipe: unresolved question, numbered sequence, contrast, or promised checklist.

## Safe Zones

- Avoid critical text in the bottom 10%.
- Avoid tiny labels near the top-right and bottom-right corners.
- Keep titles readable at phone preview size.

## Copy Tone

- Concrete, slightly conversational, not "AI product launch" voice.
- Prefer verbs and nouns from the topic.
- Keep titles short; move detail to bullets, labels, or captions.
- Use Chinese punctuation naturally.
